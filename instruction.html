Кино шаблон Kino

Адаптивный кино шаблон для DLE 13.x (младшие версии по запросу)

version: 1.0 Dle: 13.0 - 13.3 charset: UTF-8
  • Кроссбраузерная поддержка. Chrome, Firefox, Opera, IE9+
  • Крутой адаптивный дизайн.
  • SEO оптимизированный код.
  • Удобное разделение контента на главной.
  • Ajax переключение вида кратких новостей с запоминанием выбора.
  • Трейлер в ajax окне в кратких новостях.
  • Ajax закладки (данная функция доступна даже гостям, после регистрации закладки перенесутся в их профиль).
  • Подсчет колличество закладок для гостей и пользователей.
  • Вывод закладок в удобном окне.
  • Мгновенное удаление и добавление закладок в окно для вывода.
  • Крутая фишка как у GOOGLE - Голосовой поиск для DLE.
  • Вывод популярных запросов на сайте в удобном блоке.
  • Вывод добавленных коллекций в удобном окне.
  • Стильный прогресс бар в стиле youtube.
  • Удобное фиксирование шапки при скроллинге: когда скроллим вниз - скрывается, вверх - появляется.
  • Большое выпадающее меню в шапке, где можно разместить все жанры, года и тп.
  • Большой слайдер на главной с максимальной информацией о фильме.
  • Оригинальный слайдер в категориях - его фон меняется из постеров.
  • Готовое оформление под модуль фильтра от LazyDev.
  • Готовое оформление под модуль коллекций от TeraMoune.
  • Рейтинг в виде звезд с выводом общей оценки по 10 бальной шкале.
  • Рейтинг (интерес пользователей к фильму).
  • На постерах есть кнопки добавления в закладки, кнопка трейлера, оценка пользователей, интерес пользователей. Также на постерах при наведении есть иконка, наведя на которую появляется описание фильма
  • Человеко понятные даты.
  • Вывод последних обновлений на сайте в удобном окне (перейдя по ссылке все обновления пользователь попадает на страницу со всем обновлениями группированными по датам).
  • На главной для сериалов используется блок с обновлениями по датам.
  • Карусель на главной используется ajax табы
  • Вывод колличества фильмов в вашей базе.
  • Контент на главной использует ajax табы для вывода по категориям фильмов.
  • В боковой панели сделан вывод случайного фильма на ajax.
  • Место для вывода seo описания как на главной.
  • Очень круто сделаны рейтинга для кинориума,кинопоиска и imdb, вам нужно лишь добавить значения в доп. поля и шаблон автоматически сделает красивую шкалу на основе этих цифр.
  • Подгрузка новостей по нажатию кнопки на Ajax.
  • Вывод блока с просмотренными фильмами.
  • Вывод рейтинга пользователя в минипрофиле, если рейтинг отрицательный никнэйм и диаграмма рейтинга окрашивается в красный цвет, если положительный то в зеленый.
  • Страница с просмотром фильма сделана максимально удобной.
  • Оформление блока для вывода актеров (использовать модуль или иной способ вывода).
  • Встроена крутая-адаптивная галлерея fancybox.
  • Списки для гостей и пользователей (с помощью которых можно организовать списки по типу: буду смотреть, смотрю и тп)
  • Каждому фильму можно задать свой фон через дополнительное поле, либо можно использовать статичную картинку.
  • Для вывода плееров используются вкладки.
  • Включение/выключение света для удобного просмотра фильма.
  • Вывод попурярных фильмов из просмариваемой категории,как в самой категории так и в полной новости.
  • Оформление для модуля LinkEnso - модуль кольцевой перелинковки от LazyDev.
  • Если не загружен аватар, то показывается аватарка с случайным цветом, в которую подставляется первая буква имени комментатора.
  • Если рейтинг комментария отрицательный, диаграмма рейтинга окрашивается в красный цвет, а сам комментарий скрывается под спойлер.

Установка Шаблона

Установка для DataLife Engine 13.0 и выше
  • Шаг первый

    Загрузите папку engine на свой сервер.

  • Шаг второй

    Загрузите папку с шаблоном kino-dark или kino-light на свой сервер.

  • Шаг третий

    Установить через систему плагинов XML файлы из папки plugins.

  • Шаг четвертый

    Создайте дополнительные поля (или загрузите готовый файл xfields.txt в папку engine/data)

    • Название поля - poster
      Описание поля - Постер
      Тип поля - Загружаемое изображение
    • Название поля - sliderbg
      Описание поля - Задний фон слайдера
      Тип поля - Загружаемое изображение
    • Название поля - screens
      Описание поля - Скриншоты
      Тип поля - Загружаемая галерея изображений
      Установите галочку - Создать уменьшенную копию
      Размеры уменьшенной копии - 300
    • Название поля - reasonedit
      Описание поля - Причина редактирования/обновления
      Тип поля - Одна строка
  • Шаг Пятый

    Откройте файл .htaccess найдите строку RewriteEngine On и споле нее вставьте следующее (необходимо для работы модулей):.

    									
RewriteRule ^mylists/(.*?)/page/([0-9]+)(/?)+$ index.php?do=mylists&listname=$1&cstart=$2 [L]
RewriteRule ^mylists/page/([0-9]+)(/?)+$ index.php?do=mylists&listname=0&cstart=$1 [L]
RewriteRule ^mylists/(.*?)(/?)+$ index.php?do=mylists&listname=$1 [L]
RewriteRule ^mylists(/?)+$ index.php?do=mylists&listname=0 [L]

RewriteRule ^collections/([0-9]+)-(.*)/page/([0-9]+)(/?)+$ index.php?do=collections&id=$1&cstart=$3 [L]
RewriteRule ^collections/([0-9]+)-(.*)(/?)+$ index.php?do=collections&id=$1 [L]
RewriteRule ^collections/favorites(/?)+$ index.php?do=collections&action=favorites [L]
RewriteRule ^collections/favorites/page/([0-9]+)(/?)+$ index.php?do=collections&action=favorites&cstart=$1 [L]
RewriteRule ^collections/page/([0-9]+)(/?)+$ index.php?do=collections&cstart=$1 [L]
RewriteRule ^collections(/?)$ index.php?do=collections [L]

RewriteRule ^notice(/?)+$ index.php?do=notice [L]
  • Шаг шестой

    Загрузить файл install_notion.php на свой сервер и запустить его.

  • Шаг Седьмой

    Создайте дополнительную страницу с такими параметрами:

    • Название: - tops
      Описание: - Топ кино (Либо любое другое)
      Текст страницы: - Топ кино по мнению пользователей (Либо любое другое)
      Использовать шаблон: - mod-tplfiles/vendor/top100
  • Шаг Восьмой

    Откройте файл /engine/classes/js/dle_js.js найдите и закомментируйте

Работа с шаблоном

Структура файлов шаблона

  • Описание

    Описание файлов в папке mod-tplfiles

    Папка vendor содержит tpl файлы блоков верстки таких как:

    • carousel.tpl - вывод карусели на главной и категориях
      footer.tpl - подвал шаблона
      header.tpl - шапка шаблона
      main.tpl - центральный блок с контентом
      main-info.tpl - вывод блока с информацией количестве новостей в базе сайта
      search.tpl - блок с поиском и популярными запросами
      second-menu.tpl - дополнительное меню
      sidebar.tpl - боковая панель
      slider.tpl - большой слайдер на главной
      top100.tpl - вывод топов новостей

    Папка custom содержит tpl файлы шаблоном подключаемых для вывода в {custom}:

    • custom-backgroundcarousel.tpl - вывод новостей в большой слайдере каруселью
      custom-backgroundslider.tpl - вывод новостей в большом слайдере
      custom-carousel.tpl - вывод новостей в карусели на главной и категориях
      custom-favitem.tpl - вывод новостей в блоке с закладками
      custom-notice.tpl - вывод новостей в блоке просмотренного
      custom-random.tpl - вывод новостей в блоке случайного фильма
      custom-searchpopular.tpl - вывод новостей в блоке популярных запросов
      custom-top10item.tpl - вывод новостей в блоке топ 10
      custom-top100item.tpl - вывод новостей в блоке топ 100
      custom-update.tpl - вывод новостей в блоке обновлений на главной
      custom-widgetpopular.tpl - вывод новостей в боковой панели блок популярное
      custom-widgetpopularfilm.tpl - вывод новостей в боковой панели блок популярные фильмы
      custom-widgetsoon.tpl - вывод новостей в блоке скоро
  • Важно

    Шаблон использует для вывода информации {custom}. В данном тэге в шаблоне везде прописано отключение кэша, для уменьшее нагрузки рекомендуется включить кэш ( более подробнее Вы можете ознакомится в онлайн документации к CMS DLE - https://dle-news.ru/extras/online/index.html?startnews.html )

  • Разная информация

    В настройках DLE укажите:

    • Настройка параметров скрипта / Новости: Количество новостей на страницу - 15
      Настройка параметров скрипта / Новости: Количество статей на страницу в результатах поиска - 15
      Настройка параметров скрипта / Новости: Количество похожих новостей - 6 (если используете стандартный вывод похожих новостей)
      Настройка параметров скрипта / Новости: Тип рейтинга публикаций - Оценка
      Настройка параметров скрипта / Комментарии: Тип рейтинга комментариев - 'Нравится' или 'Не нравится'

    Для ускорения работы шаблона по максимуму создавайте минифицированные файлы:

    • js - http://beautifytools.com/javascript-minifier.php
      css - http://beautifytools.com/css-minifier.php
    • Настройте кэширование
    • Более подробнее прочтите: https://dle-news.ru/blog/1723-uvelichivaem-skorost-zagruzki-sayta.html
    • Более подробнее прочтите: https://8dle.ru/blog-8d/6584-v-pogone-za-pagespeed-optimizaciya-dle-nachalo.html
    • Более подробнее прочтите: https://pafnuty.name/others/133-pagespeed-2.html

    Для модуля DLE Filter v1.2.4 - модуль фильтра для DataLife Engine от LazyDev же в шаблоне все подключено, данный шаг пропустите.

Модули в шаблоне

Инструкция по настройке модулей (ВАЖНО В ШАБЛОНЕ УЖЕ ВСЕ ПРОПИСАНО, НИЖЕ ПОЯСНЯЕТСЯ ТОЛЬКО ПРЕДНАЗНАЧЕНИЕ И ИСПОЛЬЗОВАНИЕ МОДУЛЕЙ И ХАКОВ)

  • Хак "Общая оценка рейтинга"

    Данный хак выводит общую оценку рейтинга тэгом {rating_nums}.

  • Модуль "Закладки для пользователей и гостей"

    Модуль, который заменяет стандартный обработчик кнопки добавления в закладки.

    • Закладками могут пользоваться гости. Адрес страница закладок такой же как у зарегистрированных: example.com/favorites/
    • Возможность ограничить количество закладок для каждой из групп пользователей
    • При включенном кешировании в списке коротких новостей отметка не убирается
    • Более гибкая возможность настройки шаблона
    • Отзывчивый интерфейс. При клике на кнопку она тут же меняет свой вид.

    Изминения параметров происходит в файле engine/mods/favorites/config.php

  • Хак "Выбор количества новостей"

    Этот хак позволит посетителям самим выбирать, какое количество новостей будет отображаться на страницах сайта..

    Настройка колличества вывода новостей производится в файле engine/modules/news_number.php. Важно первое вхождение числа новостей должно строго совпадать с парметром в Настройка параметров скрипта / Новости: Количество новостей на страницу.

  • Хак "Вывод общего количества новостей"

    Данный хак выводит общее количество новостей на сайте тэгом {allcount}.

  • Хак "Топ новостей в категории"

    Данный хак выводит топ новостей в той категории которую просматривается пользователь {custom thiscat ...}.

  • Модуль "YouWatch"

    Данный модуль выводит новости которые уже просматрел пользователь.

    • Для подключения модуля в fullstory.tpl прописывается {include file="engine/modules/notice.php?newsid={news-id}"}

    • Что бы задать Лимит (по умолчанию 5 новостей) прописать {include file="engine/modules/notice.php?newsid={news-id}&limit=10"}

      Где 10 - количество новостей которые будут сохранятся.

    • Для того что вывести то что смотрел пользователь в нужном месте вставить [senpainoticeme]{include file="engine/modules/show.notice.php"}[/senpainoticeme]

      Теги :

      [senpainoticeme]*текст*[/senpainoticeme] - выведут текст внутри них если есть новость которую посещал юзер, иначе скроет текст.

      [not_senpainoticeme]*текст*[/not_senpainoticeme] - выведут текст внутри них если нету новостей которую посещал юзер, иначе скроет текст.

  • Модуль "Collections" (Разработчик TeraMoune)

    Подробную информацию по данному модулю Вы можете получить на странице модуля https://github.com/TeraMoune/Collections-DLE

  • Хак "Быстрый поиск"

    Хак через систему плагинов заменяет стандартный быстрый поиск на более функциональный. Шаблон для вывода search_popup.tpl

  • Модуль "Cписки для пользователей и гостей"

    Модуль позволяет организовать на своем сайте систему списков по типу: буду смотреть,смотрел и тп

    Откройте в браузере ссылку вашсайт/admin.php?mod=mylists — так вы попадете в настройки модуля. Чтобы вывести ссылку на модуль в меню «Сторонние модули» нажмите кнопку внизу справа в настройках модуля.

    • Ссылка на все закладки: /mylists/
    • Ссылка на конкретный раздел: /mylists/:URL_раздела_указанный_в_настройках
    • Вывод кол-ва новостей в конкретном разделе: добавьте к необходиимому div или span - class="mylists-count" data-id="2", * Вместо 2 укажите id нужного вам подраздела, * Чтобы вывести кол-во всех новостей в каждом подразделе укажите data-id="0"
    • Вывод блока, для быстрого внесения новости в подраздел: class="mylists-switch" data-id="{news-id}" * Вставьте данный код в нужном месте в шаблонах fullstory.tpl и shortstory.tpl, * {news-id} это тег выводящий id новости, но если вы хотите вывести блок переключения для конкретной новости в любом месте, то пишете data-id="ID_новости"
    • Вывод тега h1 в любом шаблоне: [aviable=mylists]{mylists-h1}[/aviable]
    • Вывод переключателя разделов в любом шаблоне: [aviable=mylists]{mylists-tabs}[/aviable]
    • ВАЖНО: ЗАЙДИТЕ В НАСТРОЙКИ МОДУЛЯ, НАЙДИТЕ ПОЛЕ - СПИСОК РАЗДЕЛОВ И ПРОПИШИТЕ СЛЕДУЮЩЕЕ, ЕСЛИ НЕ ПРОПИСАНО:
      1|Смотрю|smotrju
      2|Буду смотреть|budu-smotret
      3|Уже смотрел|uzhe-smotrel
  • Модуль "Архив обновлений"

    Модуль последних обновлений/редактирований новостей. Выводит последние отредактированные(обновленные) статьи в нужной части и разделах сайта, при необходимости, с возможностью "уведомления" для пользователей, были ли обновление за время отсутствия которые он мог пропустить (с возможностью подсчитать кол-во обновлений). Присутствует своя страница модуля - архив последних обновлений, все обновление за указанный период времени.

    • В нужном месте подключить модуль: {include file="engine/modules/mymod_NotiON.php"}
    • Вход в админ панель: http://site/admin.php?mod=mymod_NotiON
    • Файлы шаблонов:
      Noti_m.tpl - - в файле произвдитсья основаня настройка формления окна перечисления статей. Имеет основной тег - {tag_notiMain} отвечает за вывод статей следующего файла шаблона.
      Noti_ml.tpl - файл офрмления статей. Имеет теги:
      {tag_editdate} - неформатированная дата обновления (для скрипта уведомлений)
      {tag_reason} - прична обновления\редактирования (см. в адмичасти)
      {tag_title} - заголовок обновления\редактирования (см. в адмичасти)
      {tag_link} - ссылка на статью
      {tag_date} - дата изменения статьи
      Noti_full.tpl - шаблон отдельной траницы модуля - архива обновлений\редактирований. Имеет один осовоной тег - {tag_noti} отвечает за вывод статей следующего файла шаблона
      Noti_fl.tpl - - шаблон оформления статей на сранице архива. Имеет теги:
      {tag_reason1} - прична обновления\редактирования (см. в адмичасти)
      {tag_title1} - заголовок обновления\редактирования (см. в адмичасти)
      {tag_link1} - ссылка на статью
      {tag_date} - дата изменения статьи
  • Модуль "LinkEnso - кольцевая перелинковка" (Разработчик LazyDev)

    Подробную информацию по данному модулю Вы можете получить на странице модуля https://readme.lazydev.pro/linkenso.html
